## Variables

1. `TG_BOT_TOKEN`  - Get bot token from @BotFather
2. `APP_ID`        - From my.telegram.org
3. `API_HASH`      - From my.telegram.org
4. `RemoveBG_API`  - From [RemoveBG](https://www.remove.bg/b/background-removal-api)
